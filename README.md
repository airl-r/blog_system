# Blog System

# 个人博客系统（基于springBoot的项目）

## 来源

黑马程序员

## 版本

java 8

spring boot 2.6.6

mysql 8.0

redis 3.2

## 访问路径
端口号：8080

http://localhost:8080/

## 注意

运行程序前需要打开mysql服务器和redis服务器

虽然包含很多JavaScript的东西，但是那些都不需要我们自己编写！
